## bm.md：更强大的 Markdown 排版工具，一键适配微信公众号、网页与图片

- 适合自媒体创作者快速排版微信公众号文章
- 适合开发者在网页或图片中展示 Markdown 内容
- 支持一键导出为图片，方便社交分享
- 提供多种排版样式，满足不同场景需求
- 集成 MCP 协议，便于开发者扩展功能
- 支持 REST API，实现自动化内容处理

## NORMAL-EX/LetRecovery: Windows 系统重装与恢复工具

- 适用于需要重装或恢复 Windows 系统的用户
- 支持多种镜像格式，如 WIM、ESD 和 GHO
- 可在线下载系统镜像并进行快速安装
- 提供引导修复、驱动导入和无人值守安装功能
- 包含磁盘管理与硬件信息查看工具

![NORMAL-EX/LetRecovery: Windows 系统重装与恢复工具](/images/20260112-8021wtts4e8u8d8.png)

🔗 [访问链接](https://github.com/NORMAL-EX/LetRecovery)
